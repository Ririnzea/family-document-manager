üî• Firebase Setup untuk Family Document Manager

Info: Setup ini akan membantu Anda mengkonfigurasi Firebase untuk sync cloud dan akses multi-device.

Step 1: Buat Firebase Project

1. Klik tombol di bawah untuk membuka Firebase Console

2. Klik "Create a project" atau "Add project"

3. Nama project: family-document-manager

4. Aktifkan Google Analytics (opsional)

Buka Firebase Console

Step 2: Setup Authentication

1. Di Firebase Console, pilih "Authentication"

2. Klik "Get started"

3. Pilih tab "Sign-in method"

4. Aktifkan "Email/Password"

Step 3: Setup Firestore Database

1. Pilih "Firestore Database"

2. Klik "Create database"

3. Pilih "Start in test mode"

4. Pilih lokasi server (Asia-Southeast1 untuk Indonesia)

Step 4: Setup Storage

1. Pilih "Storage"

2. Klik "Get started"

3. Pilih "Start in test mode"

Step 5: Get Firebase Config

1. Pilih "Project settings" (ikon gear)

2. Scroll ke bawah, klik "Add app" ‚Üí Web (ikon </>)

3. Masukkan nama app: Family Document Manager

4. Copy konfigurasi Firebase dan paste di form di bawah:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId} { allow read, write: if request.auth != null && request.auth.uid == userId; } match /familyMembers/{memberId} { allow read, write: if request.auth != null && request.auth.uid == resource.data.userId; allow create: if request.auth != null && request.auth.uid == request.resource.data.userId; } match /documents/{documentId} { allow read, write: if request.auth != null && request.auth.uid == resource.data.userId; allow create: if request.auth != null && request.auth.uid == request.resource.data.userId; } } }
